# Analysis Brief Templates

Copy only the form that matches the weak point. Empty fields are useful only if they force a real analytical move.

## 1. Analytical question brief

```text
Topic:
Audience:
Decision or use:
Question type: descriptive / diagnostic / explanatory / prognostic / decision / comparative
Working question:
Boundaries: object, period, segment, geography, exclusions
Sufficient answer:
What is outside scope:
What would make us reframe the question:
```

Quality checks:

```text
Does the question point to needed evidence?
Does it avoid a built-in answer?
Does it name the decision or use?
```
